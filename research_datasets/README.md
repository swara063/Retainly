# Research datasets

Place the following CSV files here to run the benchmark-dataset comparison:

- `research_datasets/ibm_hr_attrition.csv`
- `research_datasets/saudi_employee_attrition.csv`

If a file is missing, the comparison script will tell you which one to download and where to place it.

Common sources:
- IBM HR Analytics Employee Attrition & Performance (Kaggle)
- Saudi Employee Attrition Dataset (Mendeley Data)
