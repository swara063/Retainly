# Evaluation Plan

## Technical evaluation

- Accuracy
- Precision
- Recall
- F1-score
- ROC-AUC where available
- Confusion matrix
- Model leaderboard comparison

## Explainability evaluation

The system reports top predictive features using permutation importance. This supports interpretability and helps reviewers understand why a model is making predictions.

## Fairness evaluation

The project evaluates group-level differences in prediction behavior and error rates.

## Software evaluation

- API usability
- End-to-end execution correctness
- Dashboard clarity
- Report generation
- Code modularity
- Reproducibility using sample data
